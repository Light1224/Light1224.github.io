Falsification and Reproducibility

This post is the credibility gate: try to break the signal, then verify deterministic reruns.

Robust projects survive hostile checks. This stage uses a deliberate stress sequence: randomization nulls, estimator swap tests, and reproducibility audits. If conclusions survive only one handcrafted run, they are not treated as reliable.

Randomization falsification

Time-shuffle is the strongest kill switch in the outputs: structure collapses under temporal randomization. Stock-shuffle is less absolute, which is informative in strongly coupled markets where some global structure persists even under cross-sectional reassignment.

The figure sequence moves from legacy null intuition to phase-23 distribution diagnostics.

Estimator robustness

RS-versus-GK reruns preserve the main conclusions: comparable effective signal counts and only modest concentration shifts. This lowers dependence on any single estimator definition.

Reproducibility hardening

The deterministic double-run audit passes with fixed seed and matching inventories/hashes in documented checks. This confirms that the full project can be re-executed without hidden manual edits.

Blunt summary

The result is not a perfect-pass narrative. It is a realistic profile: strong temporal structure, robust systemic signal, nuanced null behavior in edge cases, and reproducible packaging.
