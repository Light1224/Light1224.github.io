Dictionary

This dictionary is shared across all pages in this archive.

Core terms used across pages

  • Systemic mode / PC1: the leading eigenmode; broad market-volatility co-movement backbone.
  • Secondary mode / PC2: next retained structural mode; often sector-loaded and less stable.
  • Signal modes: eigenmodes above MP upper bound in the selected setup.
  • Noise bulk: eigenvalue region broadly consistent with random-matrix baseline.
  • Factor energy: squared magnitude aggregate of retained factor coordinates.
  • Principal angle: geometric rotation measure between subspaces across windows.
  • Subspace overlap: directional alignment measure between current and reference subspaces.
  • Reconstruction fidelity: closeness of filtered/low-rank matrix to raw matrix.
  • Regime: contiguous segment detected from composite signal under penalty constraints.
  • Strict-causality predictive setup: forward targets + walk-forward signal construction + train-only thresholding.
  • Falsification test: randomization-based destruction test to assess whether structure is genuine.
  • Audit-grade reproducibility: deterministic rerun with matching artifacts and hash checks.

Interpretive shorthand used throughout

  • Anchor: stable component treated as structural.
  • Warning light: unstable but informative component that can flag state change.
  • Filter, not miracle: denoising helps, but does not perfectly recover full structure.
  • Event strength over magnitude fit: good classification can coexist with weak linear R².
