# Landing Page

Technical project index focused on systems, machine learning infrastructure, and quantitative research workflows.

## Scope

- Structured project logs
- Implementation notes and build decisions
- Validation summaries and outcomes
- Repository-backed technical documentation

## Navigation

- `PROJECTS` for all published project pages
- `CV` for resume and background summary
- `EMAIL`, `GITHUB`, `LINKEDIN`, `CODEFORCES` for direct external links

## Current Status

- Homepage structure active
- Theme toggle active (light/dark)
- Static generation pipeline active
- Deployment artifacts ready from rebuild script

## Notes

This homepage is a neutral landing surface. Content modules can be expanded incrementally through source files under `content/` and regenerated via the build script.
